FUN3D calculates a functional using the following form\+: $ f = \sum_i (w_i *(C_i - C_i^*) ^ {p_i}) $ ~\newline
 Where\+: ~\newline
 $f$ \+: Functional ~\newline
 $w_i$ \+: Weighting of function ~\newline
 $C_i$ \+: Function type (cl, cd, etc.) ~\newline
 $C_i^*$ \+: Function target ~\newline
 $p_i$ \+: Exponential factor of function ~\newline
