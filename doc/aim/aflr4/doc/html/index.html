AFLR4 AIM Overview

A module in the Computational Aircraft Prototype Syntheses (CAPS) has been developed to interact with the unstructured, surface grid generator AFLR4 [1] [2].

The AFLR4 AIM provides the CAPS users with the ability to generate "unstructured, 3D surface grids" using an "Advancing-Front/Local-Reconnection (AFLR) procedure." Only triangular elements may be generated, with planned future support of quadrilateral elements.

An outline of the AIM's inputs, outputs and attributes are provided in AIM Inputs and AIM Outputs and AIM Attributes, respectively. The complete AFLR documentation is available at the SimCenter.

Details of the AIM's sharable data structures are outlined in AIM Shareable Data if connecting this AIM to other AIMs in a parent-child like manner.
