CorsairLite
-----------

Primary Developer: Cody Karcher -- ckarcher@mit.edu

This optimization package has been reduced in capability (hence "Lite") for
to keep the ESP distribution size manageable. The Python source can be found
in $ESP_ROOT/pyESP/corsairlite. Testing code and examples can be found in
$ESP_ROOT/CAPSexamples/corsairlite.
